# Cinema-Booking-web-app---backend

This repository contains the backend implementation of the Cinema Booking Web Application.  
It provides RESTful APIs to handle authentication, movie management, hall and show scheduling, seat bookings, and user reservations.

## Features
- User authentication using JWT
- Role-based access control (Admin / User)
- Movie management (Add, Update, Delete, View)
- Hall and showtime management
- Seat booking and reservation handling
- Secure API endpoints

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- dotenv
